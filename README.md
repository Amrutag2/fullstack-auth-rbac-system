# Full Stack Authentication & RBAC System

## Backend Setup

Navigate to backend folder:

```bash
cd backend-auth-rbac
```

Run backend application:

```bash
./mvnw spring-boot:run
```

Backend runs on:

```bash
http://localhost:8080
```

---

## Frontend Setup

Navigate to frontend folder:

```bash
cd frontend-auth-rbac
```

Install dependencies:

```bash
npm install
```

Run frontend:

```bash
npm run dev
```

Frontend runs on:

```bash
http://localhost:5173
```
